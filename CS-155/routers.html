Routers

What is a Router?


A router is a device that receives data and sends data
to different computer networks. A router links different networks together and
makes sure the data reaches the right IP address. It can
also allow multiple devices to share a single internet connection.


How does a Router work?

A router figures out where to send data by looking at the ip address and checking it's ip address. Then they use a set of rules to indentify the most efficent route for the data to travel.

  • Static routing - Updates manually for small networks.
  • Dynamic routing - Automatically updates for big networks.

Functions of a Router

  • Forwarding - Takes incoming packets and decides wheere to send it.
  • Routing - Choosees the best path for data to travel.
  • Network Address Translation (NAT) - Translates private IP addresses to public IP addresses.
  • Security - Protects against online threats.
  • VPN Connectivity - Provides Secure Access to networks.
  • Bandwidth Management - Controls data.
  • Monitoring & Diagnostics - Tracks Traffic and helps troubleshoot networks.

7 types of Routers

  1. Broadband Routers - Connect multiple computers to the internet.
  2. Wireless Routers - Creates Wi-fi networks.
  3. Wired Routers - Connect multiple devices using Ethernet cables.
  4. Edge Routers - protects your internal networks.
  5. Core Routers - Operates within networks.
  6. Virtual Routers - A software router.
  7. Portal Routers - Small devices creating private Wi-fi networks.
