IP Address

What is an IP Address?
an Internet Protocol address is a unique identifier assigned to every device connected to
computer networks and uses the internet for communication. IP works in helping your computer access the internet.
IP address have two different versions the IP version 4, and the IP version 6. The Ip version 4 is the oldest of the two which has space for up to 4 billion IP addresses,
while the IP version 6 has space for trillions of IP addresses. There are four type of IP address public,private, static, and dynamic IP addresses.

The four types of IP addresses

  • Public - Allows devices to access the internet. i.e 203.0.113.1
  • Private - Used within a local network. i.e 192.168.1.12
  • Static - A IP that never changes.
  • Dynamic - A IP that changes over time.
